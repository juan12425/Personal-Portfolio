The Trattoria is an Italian restaurant front-end design web app made with react-redux. It contains different features such as the management of dishes in a menu through the collection of state information that is subsequently displayed in other parts of the app. Also, it contains the possibility to retrieve menu information from a pre-built API (not available directly in GitHub).

Covid-19 analysis

Covid-19 analysis is a data science project developed with Julia and based on the MIT course 18.S190. Here I worked modeling the mechanics of a pandemic through different simulations and data analysis to create a simple machine learning model with multivariable calculus using the “gradient descent” technique. This was done by adapting an SIR system of differential equations to experimental data generated previously.

Controlled Enviroment

This is a robotics and software project intended to create a controlled environment for diversity of experiments. The photos show different parts of the circuit.

About the details, I developed the project in a way that the whole system could have a user input, for example controlling the temperature with lights through relays using an internet connection (see ESP board), and an output shown both on a web page and on a LCD screen about the data collected from different kind of sensors, consider humidity and alcohol ones (for bacteria use).
